# 📚 Simple Java Library System

This is a beginner-friendly **Java console application** that simulates a Library Management System using **Java concepts**.  
The project includes essential features such as **user signup/login, listing books, borrowing and returning books, and a wallet-based deposit system**.

This project is designed to demonstrate:
- Basic OOP (classes, objects, methods)
- Simple data handling using ArrayLists
- Console-based input/output
- Clean, understandable code structure ideal for learning Java

It includes:
- ✔ 10 preloaded books  
- ✔ User ID + password login  
- ✔ Borrow & return book functionality  
- ✔ Wallet balance deduction + refund  
- ✔ Clean 3-file structure (Book, User, LibraryApp)
